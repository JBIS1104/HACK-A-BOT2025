download_imx500_rpk_model​

download_imx500_rpk_model(model_file: str, save_dir: str) ‑> str | None

Download a model file from the Raspberry Pi IMX500 models repository.

Args:

  • model_file (str): The name of the model file to download.
  • save_dir (str): The local directory path where the downloaded file will be saved.

Raises:
RuntimeError: If the download fails due to a non-200 HTTP status code.

Returns: (str | None)
The path to the downloaded model file.
