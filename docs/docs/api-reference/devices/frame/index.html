Frame​

Frame(timestamp: str, image: numpy.ndarray, image_type: IMAGE_TYPE, width: int, height: int, channels: int, detections: Union[Classifications, Detections, Poses, Segments, Anomaly], new_detection: bool, fps: float, dps: float, color_format: COLOR_FORMAT = 'RGB', input_tensor: Optional[numpy.ndarray] = None, roi: Optional[ROI] = None)
Represents a frame in a device stream.

with device as stream:
    for frame in stream:
        ...

The frame object has the following available elements:

Attributes​

  • timestamp: str    The timestamp of the frame.

  • image_type: IMAGE_TYPE    Specification of what is visualised in frame.image.

  • width: int    The width of the frame.

  • height: int    The height of the frame.

  • channels: int    The number of channels in the frame.

  • new_detection: bool    Flag if the provided detections are updated or an old copy.

  • fps: float    The frames per second of the video stream.

  • dps: float    The detections per second in the video stream.

  • color_format: COLOR_FORMAT    The color format of the frame. Defaults to RGB.

  • roi: Optional[ROI]    Relative ROI of the input tensor. Defaults to None.

Properties​

  • image: numpy.ndarray    The image data of the frame.

  • detections: Union[Classifications, Detections, Poses, Segments, Anomaly]    The detections in the frame.

  • input_tensor: Optional[numpy.ndarray]    The input tensor of the frame. Defaults to None.

display​

display(self, show_fps_dps: bool = True, cropping: Union[ROI, Tuple[float, float, float, float], ForwardRef(None)] = None, rotate: Optional[int] = None, flip: Optional[int] = None, resize_image: bool = False)

Display the frame with various options for visualization.

Args:

  • show_fps_dps (bool = True): If True, display the frames per second (FPS) and detections per second (DPS) on the image.
  • cropping (Union[ROI, Tuple[float, float, float, float], ForwardRef(None)] = None): The region of interest to display, either as a named tuple ROI or a tuple of four floats (left, top, width, height). Defaults to None for no cropping.
  • rotate (Optional[int] = None): The rotation code for the image. Use cv2 rotation codes:

0 or cv2.ROTATE_90_CLOCKWISE: Rotate the image 90 degrees clockwise.
1 or cv2.ROTATE_180: Rotate the image 180 degrees.
2 or cv2.ROTATE_90_COUNTERCLOCKWISE: Rotate the image 90 degrees counterclockwise.
None: No rotation.

  • flip (Optional[int] = None): The flip code for the image. Use 0 for vertical, 1 for horizontal, -1 for both, or None for no flip.
  • resize_image (bool = False): If True, resize the image to fit the display window.

Raises:
ValueError: If the cropping parameter is not a valid ROI or tuple of 4 floats.
