run_shell_command​

run_shell_command(command: str)

Run shell command with output log and checking return code.

Args:

Raises:
subprocess.CalledProcessError: If the return code of the command is non-zero.

check_dir_required​

check_dir_required(dir: pathlib.Path, required_patterns: List[str])

Check if a directory contains all the required files or directories based on the given patterns.

Args:

  • dir (pathlib.Path): The directory to check.
  • required_patterns (List[str]): A list of patterns (files or directories) that should be present.

Raises:
AssertionError: If the directory is not found at the specified path.
AssertionError: If any of the required files or directories are missing in the directory.

IMX500Converter​

IMX500Converter()
Helper class for converting a Model to IMX500 accepted format. Converts a KERAS or ONNX model to a packerOut.zip file ready for packaging.

Required dependencies are automatically installed into a dedicated virtual environment and depent on its model type:

  • KERAS models: require imx500-converter[tf] and is automatically installed in ~/.modlib/.venv-imx500-converter-tf
  • ONNX models: require imx500-converter[pt] and is automatically installed in ~/.modlib/.venv-imx500-converter-pt

Example:

converter = IMX500Converter()
converter.run(
    model_file="./path/to/model.keras",
    model_type=MODEL_TYPE.KERAS,
    output_dir="./pack"
)

run​

run(self, model_file: pathlib.Path, model_type: MODEL_TYPE, output_dir: pathlib.Path, overwrite: Optional[bool] = None)

Run the converter in its dedicated virtual environment corresponding to the model type.

Args:

  • model_file (pathlib.Path): Path to the model file to be converted.
  • model_type (MODEL_TYPE): Model type of the provided model to be converted.
  • output_dir (pathlib.Path): The directory where the converted packerOut.zip file will be saved.
  • overwrite (Optional[bool] = None): If None, prompts the user for input. If True, overwrites the output directory if it exists. If False, aborts if the directory exists.
