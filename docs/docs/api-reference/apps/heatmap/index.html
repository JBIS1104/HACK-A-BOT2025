Heatmap

Heatmap​

Heatmap(cell_size: int = 20)
Functionality to create and visualize heatmaps based on object detection output data.

Example:

from modlib.apps import Heatmap

heatmap = Heatmap(cell_size = 50)
heatmap.update(frame, detections)
